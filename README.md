# Face Detection & Recognition System

YOLOv8 Transfer Learning Based Security Application

## Setup

1. Clone the repository
git clone https://github.com/vinod-nimsara/face-detection.git
cd face-detection

2. Install dependencies
pip install -r requirements.txt

3. Download the CNN model
Download `cnn_model.h5` from Google Drive and place it in the `cnn/` folder:
[Download cnn_model.h5](https://drive.google.com/file/d/1HPVdfIbVTzfNgtNmCOO4qQJVOQYC7Sn7/view?usp=drive_link)

4. Run the app
python app.py
